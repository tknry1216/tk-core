# Go (Golang) トレンドレポート 2025-2026

## 1. 最新バージョンと主要機能

### Go 1.24 (2025年2月リリース)

- **ジェネリック型エイリアス**: 型エイリアスが定義型と同様にパラメータ化可能に。`GOEXPERIMENT=noaliastypeparams`で無効化可能だが、Go 1.25で設定自体が削除予定
- **Swiss Tablesベースの新しいmap実装**: ランタイムのパフォーマンスがベンチマーク全体で平均**2〜3%のCPUオーバーヘッド削減**。小規模オブジェクトのメモリ割り当て効率化、新しいランタイム内部mutex実装も含む
- **ツールディレクティブ (`go.mod`)**: `go.mod`内で`tool`ディレクティブを使って実行可能な依存関係を追跡可能に。従来の`tools.go`ファイルでのブランクインポートが不要に
- **`testing.B.Loop`**: ベンチマークで`for b.Loop() { ... }`形式の新しいループ方法。`b.N`を使った従来の方法より高速でエラーが少ない
- **`runtime.AddCleanup`**: `runtime.SetFinalizer`より柔軟で効率的な新しいファイナリゼーション機構
- **`os.Root`**: 特定のディレクトリ内でファイルシステム操作を行うための新しい型
- **FIPS 140-3準拠**: Go暗号モジュールによるFIPS 140-3承認アルゴリズムの透過的実装
- **新しい暗号パッケージ**: `crypto/mlkem`（ML-KEM-768/1024）、`crypto/hkdf`、`crypto/pbkdf2`、`crypto/sha3`
- **ポスト量子鍵交換**: `X25519MLKEM768`鍵交換メカニズムがデフォルトで有効
- **Encrypted Client Hello (ECH)**: TLSサーバーでECHをサポート
- **`go:wasmexport`ディレクティブ**: GoプログラムからWebAssemblyホストに関数をエクスポート可能。WASIリアクター/ライブラリとしてのビルドをサポート
- **`testing/synctest`（実験的）**: 並行コードのテストを支援する新パッケージ。仮想化された時間で動作

### Go 1.25 (2025年8月リリース)

- **コンテナ対応GOMAXPROCS**: Linux上でcgroupのCPU帯域幅制限を考慮してGOMAXPROCSを自動調整。Kubernetes環境で特に有用
- **`testing/synctest`正式版**: Go 1.24で実験的だったパッケージがGA（一般提供）に昇格
- **実験的JSON v2 (`GOEXPERIMENT=jsonv2`)**: 新しいJSON実装。デコードが大幅に高速化され、ベンチマークで最大**10倍高速なアンマーシャリング**、多くの構造体でゼロヒープアロケーション
- **実験的GC「Green Tea」**: 大量の小規模オブジェクトを生成し多コアCPUで動作するプログラム向けに最適化された新しいガベージコレクタ。GCオーバーヘッドを**10〜40%削減**
- **Flight Recorder**: `runtime/trace.FlightRecorder`による軽量なインメモリリングバッファで実行トレースを記録
- **DWARF 5デバッグ情報**: コンパイラとリンカがDWARF v5を生成。デバッグ情報のサイズ削減とリンク時間の短縮
- **スタック上のスライス割り当て改善**: コンパイラがより多くの状況でスライスのバッキングストアをスタックに割り当て可能に
- **`go.mod`のignoreディレクティブ**: goコマンドが無視すべきディレクトリを指定

## 3. 言語機能の進化

### Range Over Function Types（Go 1.23〜）

Go 1.23で導入されたイテレータ機能。関数をrangeループで反復可能に。

```go
// Push Iterator（標準）- yieldコールバックを受け取る関数
func(func(V) bool)
func(func(K, V) bool)

// 使用例
for v := range myIterator {
    // ...
}
```

- **Push Iterator**: yield関数を引数に取り、値をプッシュする標準形式
- **Pull Iterator**: 呼び出すごとに次の値を返す形式
- コンテナ型は慣例として`All`メソッドを提供することが推奨
- **Generics + Iterators**: Go 1.18のジェネリクスとGo 1.23のイテレータを組み合わせることで、`Map`や`Filter`関数の実装が容易に

### ジェネリクスの成熟

- Go 1.24のジェネリック型エイリアスにより、ジェネリクスの表現力がさらに向上
- コミュニティ全体でジェネリクスの採用が拡大中（2024 H2調査で確認）

## 4. クラウドネイティブ、AI/ML、WebAssembly

### クラウドネイティブ

- GoはKubernetes、Prometheus、Docker、Terraform、Istioなど主要クラウドネイティブツールの基盤言語
- Uber、Netflix、Google、DataDog、HashiCorpなどが大規模に採用
- Cloudflareレーダーによると、Goは**API呼び出しの12%**を占め、Node.jsを抜いて自動化APIリクエストの最も人気のある言語に（前年の8.4%から上昇）
- ARM64ターゲットの開発者が約半数に。エッジコンピューティングとパフォーマンス最適化サーバーの成長を反映

### AI/ML

2026年はGoがAIエージェント開発の年と称される。

**主要AI/エージェントフレームワーク（2026年）**:

1. **LangChain Go** - LangChainのGo実装
2. **Google ADK (Agent Development Kit)** - GoogleのAIエージェント開発キット
3. **Firebase GenKit** - FirebaseのジェネレーティブAI開発キット
4. **Eino** - AIエージェントフレームワーク
5. **Jetify AI SDK** - AI SDK
6. **GoMLX** - 加速ML/数学フレームワーク。モデルをC-libraryやWebAssemblyにコンパイル可能
7. **GoLearn / Gorgonia** - ML統合フレームワーク

**主な動向**:

- GoはAIモデルの訓練ではなく、**推論サービスの高速・信頼性の高いデプロイ**で活用
- **Agent2Agent (A2A)** プロトコルと**Model Context Protocol (MCP)** でGoがAIエージェント構築の優先言語に
- 調査対象開発者の約**22%**がAI/LLM関連機能の開発に関与（2025-2026年）
- kServeなどGo製のモデルサービングインフラも拡大

### WebAssembly

- Go 1.24で`go:wasmexport`ディレクティブ追加。関数のWASMホストへのエクスポートが可能に
- WASIリアクター/ライブラリとしてのビルドをサポート
- Kubernetes向けポータブル拡張（オペレーター、サイドカー）やエッジシナリオでの採用拡大
- GoMLXがモデルをWebAssemblyにコンパイル可能にし、どの言語からでも推論可能に

## 6. 開発ツール・開発体験の改善

### ログ

- **`log/slog`**（Go 1.21〜）: 標準ライブラリの構造化ログパッケージ。新規プロジェクトの標準選択肢
- **sloglint**: slogのコードスタイル一貫性を強制するリンター
- 高性能用途では**Zap**と**Zerolog**が依然リード
- **Logrus**: メンテナンスモードだが安定して利用可能

### リンター

- **golangci-lint**: 事実上の標準。**100以上のリンター**を統合。Kubernetes、Prometheus、Terraformなど主要プロジェクトが採用
- **staticcheck**: 深い静的解析
- **nilaway**: nilパニックの検出
- **gosec**: セキュリティチェック

### AI開発ツール

- 55%の開発者がAI開発ツールに満足（ただし「やや満足」42%、「非常に満足」13%）
- **Cursor**エディタが急成長（Goプライマリ環境として4%のシェア）
- AI支援コーディングの急速な普及がGo開発体験を変革中

### その他の注目ツール

- **`go vet`の新testアナライザ**（Go 1.24）: テストパッケージ内の一般的なミスを検出
- **GNU Build IDの自動生成**（Go 1.24）: ELFプラットフォームでリンカが自動的にBuild IDを生成
- **Flight Recorder**（Go 1.25）: 実行トレースの軽量リングバッファ記録

## まとめ

2025-2026年のGoは、クラウドネイティブインフラの基盤言語としての地位を強化しつつ、**AIエージェント開発**という新しいフロンティアで急速に存在感を高めている。Go 1.24/1.25では、Swiss Tables mapやGreen Tea GCなどの**ランタイムパフォーマンス最適化**、**ポスト量子暗号**、**JSON v2**、**WebAssemblyエクスポート**など実用的な改善が多数導入された。ジェネリクスとイテレータの組み合わせによりコードの表現力が向上し、開発者満足度は91%と極めて高い水準を維持している。
