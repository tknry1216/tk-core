# Go言語トレンドサマリー (2026年4月)

## 1. Go 1.26 リリース (2026年2月)

Go 1.26が2026年2月10日にリリースされ、言語仕様・ランタイム・標準ライブラリに多数の改善が加えられた。

### 言語仕様の変更

- **`new`関数の拡張**: `new`関数のオペランドに式を指定できるようになり、変数の初期値を直接設定可能に
- **ジェネリクスの自己参照解禁**: ジェネリック型が自身の型パラメータリスト内で自身を参照できるようになり、より複雑なデータ構造やインターフェースの定義が可能に

### ランタイム・パフォーマンス

- **Green Tea GC のデフォルト有効化**: Go 1.25で実験的に導入されたGreen Teaガベージコレクタがデフォルトで有効に。小さなオブジェクト（< 512バイト）を8KiBスパン単位で処理し、ランダムなポインタ追跡を逐次スキャンに変換。実世界のプログラムでGCオーバーヘッドが**10〜40%削減**
- **SIMD対応（実験的）**: `GOEXPERIMENT=simd`で有効化可能な`simd/archsimd`パッケージが導入。amd64アーキテクチャで128/256/512ビットベクトル型（`Int8x16`, `Float64x8`等）をサポート。AVX512ベースのスキャンカーネルによりGCオーバーヘッドをさらに15〜20%改善
- **cgo呼び出しの最適化**: cgo呼び出しのベースラインランタイムオーバーヘッドが約30%削減
- **`ReadAll`の高速化**: 中間メモリの割り当てが削減され、約2倍高速化、メモリ割り当て量も約半分に

### ツールチェーン

- **`go fix`コマンドの刷新**: Go分析フレームワークを使用して完全に書き直され、最新のイディオムやAPIへのコード更新を自動化する数十の「モダナイザー」を搭載

### セキュリティ・暗号化

- **HPKE対応**: `crypto/hpke`パッケージが新規追加（RFC 9180準拠）。ポスト量子ハイブリッドKEMもサポート

## 2. Go 1.24 / 1.25 の主要機能（振り返り）

### Go 1.24

- **ジェネリック型エイリアス**: 型エイリアスをジェネリック型のようにパラメータ化可能に
- **`go.mod`でのツール依存管理**: `tool`ディレクティブにより、従来の`tools.go`ワークアラウンドが不要に
- **`go:wasmexport`ディレクティブ**: WebAssemblyホストへの関数エクスポートが可能に
- **ポスト量子暗号**: X25519MLKEM768鍵交換メカニズムがデフォルトで有効
- **Encrypted Client Hello (ECH)**: TLSサーバーでECHをサポート

### Go 1.25

- **実験的JSON v2実装**: `GOEXPERIMENT=jsonv2`で新しいJSON実装を利用可能
- **DWARF v5デバッグ情報**: デバッグ情報のサイズ削減とリンク時間の短縮（特に大規模バイナリ）
