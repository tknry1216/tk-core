## Go 1.26 リリース（2026年2月）

Go 1.26が2026年2月にリリースされ、言語仕様の改善・パフォーマンスの大幅な向上・開発者体験の改善が図られた。

### 言語仕様の変更

- **`new` 関数の拡張**: `new` のオペランドに式を指定可能になり、変数の初期値を直接指定できるようになった
- **自己参照ジェネリクス**: ジェネリック型が自身の型パラメータリスト内で自身を参照可能に。複雑なデータ構造やインターフェースの実装が簡素化

### パフォーマンス改善

- **Green Tea GC がデフォルト有効化**: 以前は実験的だった Green Tea ガベージコレクタがデフォルトで有効に。実際のプログラムで **10〜40%** の GC オーバーヘッド削減を達成。平均一時停止時間は高負荷時でも **100マイクロ秒未満**
- **cgo オーバーヘッド約30%削減**: ベースラインの cgo 呼び出しコストが大幅に低減
- **スタックアロケーションの改善**: コンパイラがスライスのバッキングストアをスタック上に配置できるケースが拡大し、ヒープアロケーションを回避

### 新パッケージ・標準ライブラリ

- **`crypto/hpke`**: RFC 9180 準拠の Hybrid Public Key Encryption (HPKE) を実装。ポスト量子ハイブリッド KEM もサポート
- **`simd/archsimd`** (実験的): アーキテクチャ固有の SIMD 操作を提供。amd64 で 128/256/512 ビットベクトル型を利用可能
- **`runtime/secret`** (実験的): 暗号コードで使用される一時変数の安全な消去機能を提供

### Go が AI ツーリングで選ばれる理由

2026年、Go は AI エージェント・ツーリング領域で急速に採用が進んでいる。

- **AI エージェント通信プロトコル**: Agent2Agent (A2A) や Model Context Protocol (MCP) の実装言語として Go が選好されている
