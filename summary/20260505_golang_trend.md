## Go バージョンリリース状況

### Go 1.24（2025年2月）
- **ジェネリック型エイリアス**が完全サポート
- **Weak ポインタ**（`weak` パッケージ）の導入
- **Swiss Tables** によるmap実装の刷新（ルックアップ性能10〜60%向上）
- `go build -json` フラグで構造化JSON出力に対応
- **FIPS 140-3** 暗号化コンプライアンス対応
- `go tool` が `go.mod` からツールを直接実行可能に

### Go 1.25（2025年8月）
- `go build -asan` の強化
- `go mod ignore` ディレクティブの追加
- `go doc -http` ローカルドキュメントサーバーの新設
- **コンテナ対応 GOMAXPROCS**（コンテナ環境での自動最適化）
- 実験的GC改善と **FlightRecorder** の追加

### Go 1.26（2026年2月）
- `runtime/secret` パッケージ（実験的）による秘密情報の安全な破棄
- `go mod init` で生成される go directive のデフォルト値が変更
- 2026年3月に Go 1.26.1 がリリースされ、暗号化・HTMLテンプレート・URLパッケージのセキュリティ修正を含む

## 主要トレンド

### 1. クラウドネイティブ領域での圧倒的存在感
- Docker、Kubernetes、Terraform、Prometheus などクラウドインフラの中核ツールがGo製
- プラットフォームエンジニアリング・社内開発者プラットフォーム（IDP）構築での採用が拡大
- バックエンド開発・クラウドインフラ・CLIツール開発で事実上の標準言語として定着

### 2. AI/ML インフラストラクチャへの進出
- **Ollama** の爆発的人気（GitHub 100K+スター）により、GoがAIインフラ言語として認知
- モデル訓練ではなく、サービング・オーケストレーション・AIアプリ構築に強み
- LLMツール/エージェントフレームワークがGoエコシステムで台頭

### 3. WebAssembly（Wasm）の成熟
- `GOOS=wasip1` サポート（Go 1.21以降、継続的に改善）
- サーバーレスエッジコンピューティングでの活用（Cloudflare Workers、Fastly Compute）

### 4. パフォーマンス改善
- Swiss Tables map実装によるルックアップ性能の大幅向上（10〜60%）
- Profile-Guided Optimization（PGO）の継続的改善
- 各リリースでのランタイム・GC改善

### 5. 開発者体験の向上
- `gopls`（言語サーバー）の大幅な高速化と機能拡充
- VS Code Go拡張機能の改善
- ツールチェインのエラーメッセージ・診断機能の改善
- `go telemetry` によるオプトイン利用データ収集

## まとめ

2025〜2026年のGoは、クラウドネイティブ領域での支配的地位を維持しつつ、AI/MLインフラやWebAssemblyといった新領域への展開が加速している。言語自体もジェネリクスの成熟、イテレータの標準化、パフォーマンス改善（Swiss Tables、PGO）など着実に進化を続けており、開発者満足度も高い水準を保っている。
