# Go言語トレンドサマリー (2026年4月)

## 1. Go 1.26 リリース (2026年2月)

Go 1.26が2026年2月10日にリリースされた。主な新機能は以下の通り。

### 言語仕様の変更
- **`new`関数の拡張**: 組み込み`new`関数のオペランドに式を指定可能になり、変数の初期値を直接設定できるようになった
- **`go fix`の刷新**: コードをモダンなGo機能に自動更新する新しい`go fix`実装が導入された

### パフォーマンス改善
- **cgoオーバーヘッド約30%削減**: cgo呼び出しのベースラインランタイムオーバーヘッドが大幅に軽減
- **Green Tea GCのデフォルト化**: 実験的だったGreen Teaガベージコレクタがデフォルトで有効に
- **スタックアロケーション最適化**: スライスのスタック割り当てが改善され、GCオーバーヘッドがさらに削減

### セキュリティ
- **ヒープベースアドレスのランダム化**: 64bitプラットフォームで起動時にヒープベースアドレスをランダム化し、メモリアドレス予測攻撃を困難に
- **HPKE (Hybrid Public Key Encryption)**: 新パッケージ`crypto/hpke`が追加され、RFC 9180準拠のHPKEをサポート（ポスト量子ハイブリッドKEM含む）
- **JPEG処理の刷新**: エンコーダ・デコーダがより高速かつ正確な実装に置き換え

## 2. エコシステムとコミュニティの動向

### 開発者数の成長

- セカンダリ言語を含めると**500万人超**

### ツーリング

- **AI開発支援**: Go開発者の**70%**がAIアシスタントを日常的に活用

## 3. 注目トレンド

### AIツーリングとエージェント開発
- Pythonが研究用途で優勢な一方、Goは**モデルデプロイメント・高性能推論環境**での採用が拡大
- プロダクショングレードのAIエージェント構築において、Goの並行処理モデルとスケーラビリティが評価されている
- **Go MCP SDK v1.0.0**が公式リリースされ、AI/MCPエコシステムへの本格参入

### クラウドネイティブ・DevOps
- Kubernetes、Docker等のクラウドネイティブツールの拡大に伴い、Goのエコシステムも引き続き成長
- DevOpsスタックの中核言語としての地位を強化

### IoT・エッジコンピューティング
- 低メモリフットプリントと高速実行により、IoTデバイスやエッジゲートウェイでの活用が拡大
- 並行処理モデルにより大量の同時データストリームを効率的に処理

### GPUエコシステム
- Go 1.26で58万行以上のPure Goによる**GPUエコシステム**が構築（シェーダコンパイラからGUIツールキットまで）

## 4. セキュリティの取り組み
- 頻繁なセキュリティポイントリリースを実施（2025年10月には単一リリースで10件の修正）
- Trail of Bits監査で**低重大度の問題1件のみ**という高い品質を実証
- ポスト量子暗号への対応を強化（Go 1.24でのX25519MLKEM768、Go 1.26でのHPKE）
